# AI秘書プロジェクト（最小構成）

Claude Code を使った手動運用でスタートするAI秘書プロジェクトです。
まずシンプルに動かすことを優先し、ツール連携は後から順次追加していきます。

## 運用方針

- **現在**: Claude Code を使った手動オペレーションで運用
- **今後追加予定**: Google Workspace（Gmail / カレンダー）・Chatwork との本格連携

## 最初のゴール

1. **今日の整理** — その日の予定・タスク・連絡事項を一覧化する
2. **明日の優先タスク3件の作成** — 翌日に集中すべきことを3件に絞って明日に備える

## 毎朝の確認事項

- 今日の予定を確認する
- 要返信メール・メッセージを洗い出す
- 重要な連絡事項を把握する
- 今日の優先タスク3件を決める

## 毎夕の確認事項

- 今日完了したタスクを記録する
- 未完了・持ち越しタスクを整理する
- 明日の優先タスク3件を決める
- 気になること・メモを残す

## ディレクトリ構成

```
ai-company/
├── README.md
├── reports/        # 生成されたレポートや出力ファイル
├── memory/         # エージェントの記憶・ナレッジベース
├── scripts/        # 朝夕ルーティンのテンプレート
│   ├── morning-brief.md
│   └── evening-report.md
└── .claude/
    └── agents/     # Claudeサブエージェント定義
        ├── secretary.md
        └── pm.md
```

## エージェント

- **secretary**: スケジュール管理・メール・タスク対応を行うAI秘書
- **pm**: プロジェクト管理・進捗追跡を行うAIプロジェクトマネージャー

## 運用手順

> 現時点はすべて手動運用です。ユーザーが情報をテキストで渡し、Claude Code 上でエージェントに処理させます。

### 毎朝の使い方

1. `scripts/morning-brief.md` を開く
2. 今日の予定・要返信事項・重要連絡をテキストで用意する
3. secretary に情報を渡し、morning brief を出力させる
4. 出力結果を `reports/morning-brief-YYYY-MM-DD.md` として保存する

### 毎夕の使い方

1. `scripts/evening-report.md` を開く
2. 今日の完了事項・未完了事項・気になったことをテキストで用意する
3. secretary または pm に情報を渡し、evening report を出力させる
4. 出力結果を `reports/evening-report-YYYY-MM-DD.md` として保存する

### secretary を使う場面

- 今日の予定・タスク・連絡事項を整理したいとき
- 要返信メールや重要連絡をまとめたいとき
- 今日・明日の優先タスクを決めたいとき
- メールや文書の下書きを作りたいとき

### pm を使う場面

- プロジェクトの進捗を整理・確認したいとき
- 未完了タスクや持ち越し事項を一覧化したいとき
- 遅延リスクや懸念点を洗い出したいとき
- 複数タスクの優先順位を判断したいとき

### reports/ に保存されるファイルの扱い

- ファイル名は `[種別]-YYYY-MM-DD.md` で統一する（例: `morning-brief-2026-03-14.md`）
- 過去のファイルは削除せず蓄積する（振り返りの記録として活用）
- 編集が必要な場合はファイルを直接修正する

### memory/ に保存する際の注意点

- ユーザーが明示的に「重要」「記録しておいて」と指示した内容のみ保存する
- 毎回の会話内容や日常的な整理結果は保存しない
- 継続して参照すべき判断基準・固定情報（クライアント情報、ルールなど）を保存する

## コピペ用プロンプト

### 朝用（secretary へ）

```
以下の情報をもとに、今日の morning brief を作成してください。

【今日の予定】


【要返信事項】


【重要連絡】


【今日やるべきこと】

```

### 夜用（secretary または pm へ）

```
以下の情報をもとに、今日の evening report を作成してください。

【今日完了したこと】


【未完了事項】


【気になったこと】


【明日やること候補】

```

### secretary への短い指示文

```
上記の情報を secretary として整理してください。
優先タスク3件・確認すべきこと・次にやることを出力してください。
```

### pm への短い指示文

```
上記の情報を pm として整理してください。
進捗・未完了・懸念点・明日の優先タスク3件を出力してください。
```
